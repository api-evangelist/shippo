# ShippoThis is a repo for managing machine readable artifacts for the [Shippo APIs](https://goshippo.com). The goal is to maintain and evolve an OpenAPI, Postman Collection, and APIs.json index for the [Shippo APIs](https://goshippo.com).They are a constant work in progress, and will be updated on a regular basis when new information is added, or more of the specifications are supported.Feel free to submit a pull request to evolve any of the artifacts, and leave GitHub issues if you have any questions, comments, or feedback.- [Website](https://goshippo.com)- [Developer](https://goshippo.com)- [Twitter](https://twitter.com/goshippo)- [GitHub](https://github.com/goshippo)The [APIs.json](https://github.com/api-evangelist/shippo/blob/master/apis.json) index provides links to all the relevant links for the Shippo APIs, including the OpenAPI and Postman if they are present.
